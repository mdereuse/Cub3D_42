## Launch

`make bonus && ./cub3D eval/bonus.cub`

## Commands

* w, a, s, d : directional keys
* right, left : look around
* up, down : look up/down
* c : crouch (then c again to stand up)
* space bar : jump
* f : fight
* o : open a door
* p : pick up an item
* escape : quit
